#Installation#
 
 1. Append a repository to your composer.json
	
	 	{
            "repositories": [
                {
                "type": "github",
                "url":  "https://github.com/CpanelInc/xmlapi-php.git"
                }
            ]
        }
        	
 1.  Setup package
 
 			composer require 'CpanelInc/xmlapi-php:dev-master'
